# zhcn-pgsoft-pages

## 项目简介

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面内容不针对任何具体域名或网站，仅作为静态资源存放和版本管理的用途。  
仓库名称中的 `zhcn-pgsoft-pages` 表示其内容与中文环境及通用页面归档相关。

## 目录结构

```
.
├── README.md          # 本文件
├── html/              # 存放归档的 HTML 页面文件
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/            # 页面所引用的资源文件（CSS、JS、图片等）
    ├── styles/
    ├── scripts/
    └── images/
```

## 页面归档说明

- 所有 HTML 页面均放置在 `html/` 目录下，每个文件独立成一个页面。
- 页面之间无相互依赖关系，可单独访问或部署。
- 资源文件统一存放于 `assets/` 目录，便于管理和更新。
- 页面内容可能包含示例、工具、说明等，但不指向任何特定商业或运营目标。

## 使用方式

1. 直接打开 `html/` 目录下的任意 `.html` 文件即可在浏览器中查看页面。
2. 也可将整个仓库部署到静态托管服务（如 GitHub Pages、Netlify 等）进行在线访问。
3. 如需引用资源，请确保路径指向 `assets/` 下的对应文件。

## 维护说明

- 欢迎通过 Issue 提出建议或反馈，但请注意避免添加与具体域名、营销相关的信息。
- 提交 Pull Request 前请确保页面内容符合归档主题，且不包含外部链接或诱导性内容。
